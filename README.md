# MQTT Security Layers

## Network

- UFW Firewall Rules

## Transport Layer

- Broker TLS Certficiates
- Client TLS Certificates

## Authentication Layer

- Usernames / Passwords

## Authorization Layer

- ACL / Permissions

## Application Layer

- Encrypted Payloads
